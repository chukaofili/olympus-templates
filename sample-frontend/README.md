## Installation

These instructions are for use without docker

* Make sure that [Node.js](https://nodejs.org/) is installed.
* Install Node.js modules with `yarn`:
```shell
yarn install
```

## Starting App

* To run the app, cd to the application folder.
* Start the app with `yarn`:
```shell
yarn start
```
